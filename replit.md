# ClinicalGuidelines.io - The Vascular Intelligence Project

## Overview
A static, responsive business website for a medical AI research initiative. The design follows a "Medical, Academic, and Trustworthy" aesthetic using deep navy blues, clean white space, and crisp sans-serif fonts.

## Tech Stack
- HTML5
- CSS3 (custom properties, flexbox, grid)
- Vanilla JavaScript
- Chart.js (for benchmark visualization)
- Google Fonts (Inter, Merriweather)

## Project Structure
```
/
├── index.html       # Main HTML file with all sections
├── style.css        # Complete styling with responsive design
├── script.js        # Mobile menu, chart, animations, form handling
├── package.json     # Node.js config for serve
└── replit.md        # This file
```

## Key Sections
1. **Navigation** - Fixed header with mobile hamburger menu
2. **Hero** - High-impact gradient background with CTAs
3. **Mission** - Three-card grid explaining core values
4. **Evidence** - Chart.js bar graph comparing AI vs human performance
5. **Technology** - Why GraphRAG solves hallucination problem
6. **How It Works** - 4-step process explanation
7. **Team** - Scientific leadership display
8. **Contact** - Beta access request form
9. **Footer** - Links, contact info, disclaimer

## Design System
- Primary Deep: #002D62
- Accent Blue: #007BFF
- Background Light: #F8F9FA
- Fonts: Merriweather (headings), Inter (body)

## Running the Project
The website is served using `npx serve -s . -l 5000`
